Chinese Cuisine Web App

NOV 2016 - APR 2017

DEMO

PROJECT DESCRIPTION

Chinese Cuisine Web App introduces a collection of Chinese traditional lipsmacking dishes that tickle people's culinary senses. People can use this web app to view promotions, browse dishes by category, comment and rate for each dish, filter, and submit feedback. In summary, this is a flexible template for restaurants to set up and promote their online menus quickly.

This web app was implemented using Angular JS MVC architecture, along with bootstrap framwork and HTML/CSS/Javascript scripts. The data are hosted on Firebase in JASON format. Multiple methods and tools have been used throughout the development of the project, including LESS and Sass, Grunt and Gulp, Unit testing, e2e testing, and Yo and Yomen.
